## Simple Flash Package

<p>Originally modelled after laracasts/flash</p>

#### Installation:

<p>composer.json</p>
```
"require": {
  "iadambrown/flash": "dev-master"
},

...

"repositories": [
  {
    "type": "vcs",
    "url": "https://github.com/iadambrown/flash.git"
  }
],
```
